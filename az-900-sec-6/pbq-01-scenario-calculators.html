Scenario: Which Calculator?

AZ-900 Section 6 · Scenario

A company's CTO has asked for two reports. First, the finance team wants to know how much it would cost to run three web apps, a SQL database, and a storage account in Azure per month. Second, the board wants a comparison showing how much the company would save by moving its entire on-premises data centre — 20 servers, storage arrays, and associated labour costs — to Azure.

Which tools should be used for each report?

A. Azure Cost Management for both reports
B. Azure Pricing Calculator for the monthly cost estimate, TCO Calculator for the migration savings comparison
C. TCO Calculator for both — it covers all cost scenarios
D. Azure Advisor for the monthly estimate, Pricing Calculator for the migration comparison
